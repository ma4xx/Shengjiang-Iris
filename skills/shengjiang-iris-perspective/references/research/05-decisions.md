# 05 · Decision Records

本文件记录能体现她“怎么判断”的公开决策案例。

## 决策 1：从学术编辑器一路 pivot 到 AFFiNE 的生产力方向

### 公开事实

- 2020 年开始创业，最初做学术编辑器
- 后续 pivot 到开源编辑器内核，再到文档与白板融合的生产力工具

### 暴露出的判断

- 她接受 pivot，但强调 pivot 必须沿着核心能力走
- 她不是“遇到热点就换方向”，而是沿编辑器内核这条主线不断调整叙事和市场

### 对应模型

- `Pivot 可以频繁，但主线不能丢`

## 决策 2：坚定把开源当成冷启动与全球增长入口

### 公开事实

- AFFiNE 2022 年 8 月开源
- 首周约 6000 star，后续持续放大

### 暴露出的判断

- 开源可以是极强的 attention engine
- 但 attention 必须被 Readme、文档、教程、社区和商业设计承接

### 对应模型

- `开源、内容与社区只是杠杆，不是收入本身`
- `运营不是宣发，是用户路径设计`

## 决策 3：Demo Day / Launch 走极致精细化运营

### 公开事实

- 她明确讲过自己会设计线下物料、二维码、用户流向和入口回收
- 她把开源 launch 的入口统一导回 GitHub Readme

### 暴露出的判断

- 她理解的运营不是热闹，而是“你希望用户从哪里进、在哪里停、最后做什么”
- 用户路径是可以预设的，不是完全听天由命

### 对应模型

- `运营不是宣发，是用户路径设计`

## 决策 4：反对在早期盲目追 PR 与大投流

### 公开事实

- 她公开说过，很多创业者急着做 PR 和投流，但应该先提高注册到付费转化率
- 她建议小微团队宁可牺牲一部分用户，也要先提高营收

### 暴露出的判断

- 她对 early-stage growth 的排序是：
  - 先看是否有闭环
  - 再看是否值得放大
- 她天然不相信“靠声量救转化”

### 对应模型

- `先闭环，后放大`

## 决策 5：离开 AFFiNE 后转向一人公司与合作生态

### 公开事实

- 她在 2024 年 4 月离开 AFFiNE
- 后续公开身份更偏顾问、导师、一人公司实践者

### 暴露出的判断

- 她对“扩团队”没有天然执念
- 她更相信基于信任与判断的协作网络

### 对应模型

- 这不是她公开增长方法的主模型，但会影响她对资源配置和组织方式的建议

## 决策 6：明确反对对赌协议

### 公开事实

- 她公开说过坚决反对签对赌协议

### 暴露出的判断

- 她非常重视长期生存权，不会为了短期资金牺牲结构性安全
- 她偏向在小团队阶段先保住判断权与节奏

### 对应模型

- 与增长无直接关系，但能看出她的底层价值观：
  - 先活下来
  - 先掌控节奏
  - 不为了好看接受坏交易
